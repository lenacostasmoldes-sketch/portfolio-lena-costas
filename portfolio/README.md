# Portfolio de Lena Costas

Sitio web estático (HTML/CSS/JS) preparado para publicarse en GitHub Pages.

## Publicación rápida en GitHub

1. Crea un repositorio nuevo en GitHub (por ejemplo: `portfolio-lena-costas`).
2. Sube el contenido completo de esta carpeta al repositorio.
3. Asegúrate de que la rama principal se llama `main`.
4. Ve a la pestaña **Actions** y confirma que el workflow `Deploy Portfolio to GitHub Pages` se ejecuta correctamente.
5. Cuando termine, GitHub publicará la web automáticamente en la URL de Pages del repositorio.

## Archivos clave para el deploy

- `.github/workflows/deploy-pages.yml`: despliegue automático a GitHub Pages en cada push a `main`.
- `.nojekyll`: evita que Jekyll procese el sitio.
